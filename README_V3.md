## 🔧 Configuración (primera vez solo)

### Para capturar llamadas (Zoom/Meet/Teams)

**macOS (BlackHole):**
1. Descarga BlackHole 2ch (gratis): https://existential.audio/blackhole/
2. Instala y reinicia
3. Abre "Configuración de Audio MIDI" (Spotlight → "audio midi")
4. Click "+" → "Crear dispositivo de salida múltiple"
5. Marca: tus altavoces + BlackHole 2ch
6. En Ajustes del sistema → Sonido → Salida: elige ese dispositivo
7. Vuelve a la app, pulsa "↺" y selecciona "BlackHole 2ch"

**Windows (VB-Audio Cable):**
1. Descarga VB-Audio Virtual Cable (gratis): https://vb-audio.com/Cable/
2. Instala y reinicia
3. En Sonido → Selecciona "CABLE Output" como fuente de audio
4. En la app, elige "CABLE Output"

Con esto, **TODO lo que suene en tu computadora entra al traductor**:
- Llamadas Zoom/Meet/Teams
- YouTube
- Netflix
- Podcasts
- Conversaciones en Discord
- Etc.

## 🚨 Troubleshooting

### "No puedo escuchar las llamadas de Zoom"

**Solución:** Necesitas BlackHole (macOS) o VB-Cable (Windows)
- BlackHole solo captura audio DEL SISTEMA (Zoom, YouTube, etc.)
- El micrófono solo escucha TU voz
- Usa BlackHole → tendrás ambas cosas

### "La traducción es lenta"

- TOGETHER.AI: 2–3 segundos es normal
- La app necesita: 0.5s (transcribir) + 1.5s (traducir) = 2s total
- Es el límite de las APIs públicas

### "Se ve pixelada la interfaz"

- En macOS: abre Preferencias del Sistema → Accesibilidad → Opciones de pantalla
- Desactiva "Aumentar contraste"

### "No aparece la pestaña Admin"

- Solo aparece si estás logueado como `samortelo@gmail.com`
- O si tu email está marcado como admin en la BD
